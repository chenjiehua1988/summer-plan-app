# 学习计划 App · 产品说明文档

家庭学习管理工具——给孩子规划学习周期、每日任务打卡、爸妈验收、积分奖励、统计报表。支持暑假/寒假/KET备考等长期场景，爸妈与孩子分身份使用，多端同步。

---

## 一、安装与登录

### 安装（不走应用商店）
1. iPhone 用 **Safari** 打开网址 `https://chenjiehua1988.github.io/summer-plan-app/`
2. 点底部分享按钮 →「添加到主屏幕」→ 添加
3. 桌面出现「学习计划」图标，点开全屏运行
> 安卓用 Chrome → 菜单 →「添加到主屏幕」。
> **更新后**（有新功能时）：删旧图标重新添加，或 Safari 长按刷新图标选"重新载入"，才能加载新版。

### 登录页
打开 app 未登录时显示登录页，顶部两个 Tab：

**「家长」Tab**
- 登录 / 首次创建家庭 两个子标签
- **家长登录**：填家庭名 + 家庭密码 + 选"妈妈/爸爸"角色 → 登录
  - 角色登录后**固定，不能切换**（操作记录可追溯是谁做的）
  - 想换角色：退出后用另一角色重新登录
- **首次创建家庭**：填家庭名 + 密码（至少4位）+ 选角色 → 创建
  - 创建后配偶用同一家庭名+密码登录，各自选角色

**「我是孩子」Tab**
- 填家庭名 → 点「查找家庭」→ 选自己（小修灏/小修宇）→ 点「进入」
- 不需要密码（家庭名+孩子名即身份）
- 进入后是**孩子模式**，权限受限

### 顶栏（登录后）
- **左上**：家长模式显示孩子切换器（切换查看哪个孩子）；孩子模式显示当前孩子名+年级
- **中间**：学习周期切换器 + 倒计时（如"剩 62 天"）
- **右上**：⭐积分按钮（点开积分中心）+「退出」按钮

### 底部导航
- 家长模式：📅今日 / ✅验收 / 📊统计 / 🏃生活 / ⚙️设置
- 孩子模式：📅今日 / 📊统计（隐藏验收/生活/设置）

## 二、今日页（📅）

孩子和家长都能看。显示当前周期+当前孩子今天的任务清单。

### 顶部
- 日期 + "已完成 X/Y"（Y 排除免打卡的）
- 「刷新」按钮
- 假期条：
  - 今天不是假期：家长显示「今天设为假期」「预设假期时段」；孩子显示"今天正常学习"
  - 今天是假期：显示"🏖️ 今天是假期（原因）"+ 家长有「取消假期」

### 任务清单
按标签分组显示当天任务。每个任务卡片：
- **左侧圆圈**：勾选完成（已验收的锁定不能取消，免打卡的灰色—）
- **任务名** + 科目标签（语文/数学/英语/生活）+ 自定义标签
- **状态徽章**：待验收/已验收/被打回/免打卡
- **📷 N**：有N张照片，点开查看
- **🎙 N**：有N段录音，点开播放
- **备注**：📝 显示打卡备注
- **打卡时间**：打卡 HH:MM
- **验收时间**：验收 HH:MM · 验收人
- **右侧**：积分（+N）
- **操作按钮**：
  - 未完成：「记」（打开打卡面板）
  - 待验收：「改」+「历史」
  - 已验收：「历史」（看多次打卡记录）
  - 免打卡：无按钮

### 打卡面板（点「记」/「改」）
底部弹出抽屉，包含：
- **任务名**
- **说明输入框**：填备注（可选），带出上次的备注
- **📷 拍照/选图**：可拍多张或从相册选，已选的显示为可删芯片
- **🎙 录音**：浏览器录音（开始/停止），用于背诵任务
- **🎵 选音频文件**：从手机选已有录音文件（iOS录音不灵时用）
- **已带出上次的照片/录音**：显示为芯片，点×可删（打回重做时带出，可增删改）
- **完成打卡**：提交（上传新照片/录音 + 写打卡流水）

### 假期功能
- **今天设为假期**：当天任务全部转"免打卡"，不计入完成率
- **预设假期时段**：底部面板选开始~结束日期，一次标记整段（如旅游7/10-7/15）
- **取消假期**：当天任务恢复待打卡
- 假期内不生成打卡任务，统计自动排除

## 三、验收页（✅）（仅家长）

爸妈验收孩子已完成的任务。

### 顶部
- "待验收 N · 已验收 M"
- 刷新

### 待验收区
每个待验收任务（孩子打了卡等验收的）：
- 任务名 + 科目 + 积分
- 📷 N（看孩子传的照片）/ 🎙 N（听录音）
- 备注、打卡时间
- 备注输入框（验收备注）
- **📷 补拍照片**：爸妈补传存档照
- **通过** / **打回** 按钮
  - 通过：状态变已验收，**自动加分**（一个任务只加一次）
  - 打回：状态变被打回，不加分，孩子可重做

### 已验收区
- 显示已验收任务，带「撤销」按钮
- **撤销**：状态变回"被打回"，**扣回已加的分**。孩子重做通过后重新加分（判重触发器保证不重复加）

### 已打回区
- 显示被打回的任务

### 看照片/录音
- 点📷打开照片列表 → 点单张**全屏放大**
  - 双指缩放（1-6倍）、单指平移、双击放大/还原
  - ⟳ 旋转90度、⬇ 下载到本地、‹ › 切换上下张、✕ 关闭
- 点🎙打开录音列表 → audio 控件播放

## 四、统计页（📊）

孩子和家长都能看（孩子只看自己）。

### 周期进度条
- 总天数/已过天数 + 倒计时（剩 N 天）
- 进度条

### 总览卡片
- 验收完成率（排除免打卡）
- 连续打卡天数
- 周期积分
- 假期天数
- "验收 X/Y · 免打卡 Z"

### 按标签完成率
- 学校标签完成率、自主标签完成率…
- 进度条 + X/Y

### 按科目完成率
- 语文/数学/英语/生活 各科完成率

### 近7天完成数柱状图

### 历史周期对比
- 其他周期（已归档/进行中）的完成率横向对比

### 打卡明细
- 选某天日期 → 点「查看」
- 列出该天所有打卡记录（按时间倒序）：时间、谁打的、任务名、备注、照片、录音
- 可看历史任意一天的详细打卡

## 五、生活页（🏃）（仅家长）

记录非学业生活项：运动/阅读/家务/屏幕时间。
- 选类型（横排选择卡）+ 填内容 → 记录
- 显示今日记录 + 最近7天记录

## 六、设置页（⚙️）（仅家长）

### 家庭
- 家庭名称、当前身份（角色固定不可改）

### 兑换申请
- 列出孩子的兑换申请，每条「同意/拒绝」
- 同意：扣分 + 写兑换记录 + 申请置同意
- 拒绝：申请置拒绝

### 周期类型管理
- 自定义周期类型（暑假/寒假/KET备考/日常/其他 + 可加"期中备考"等）
- 增删

### 学习周期
- 列出所有周期，可归档/删除
- 新建周期：名 + 类型 + 起止日期
- 新建时可选"从旧周期复制清单"

### 标签管理
- 自定义标签（学校/自主 + 可加"预习""拓展"等）
- 增删、改颜色

### 兑换商店目录
- 奖励项增删（看一集动画5分/玩游戏10分/1元10分/玩具200分…）
- 上下架、改积分

### 孩子档案
- 增删孩子（姓名+年级）

### 积分中心（嵌入）
- 余额、兑换商店（点兑换直接扣分）、积分流水、兑换记录

### 任务清单（当前周期·当前孩子）
- **+添加任务**：打开任务面板
- 每个任务有「改」「删除」

### 任务面板（添加/编辑）
底部抽屉，包含：
- 任务名
- 科目（横排选择卡：语数英生活）
- 用时（分钟）、积分
- 标签（多选选择卡）
- 生效日期（开始~结束，留空=整周期）
- 每周哪几天（多选 日一二三四五六，不选=每天）
- 启用/停用
- 保存 / 删除
> 任务可设"7/10起~8/20止"+"只周六日"，叠加生效。

### 账号
- 退出登录

## 七、积分中心（点顶栏⭐打开，家长孩子通用）

家长和孩子都能进（孩子看自己）。

### 余额
- ⭐ 当前积分

### 兑换商店
- 列出上架奖励（图标+名+积分）
- **家长模式**：「兑换」→ 直接扣分兑换
- **孩子模式**：「我想兑换」→ 生成申请，等爸妈同意（不直接扣分）

### 我的兑换申请（孩子模式）
- 显示申请状态：待审批/已同意/已拒绝

### 积分流水
- 最近30条加减分明细（验收通过+X、兑换-X）

### 已兑换奖励
- 兑换记录

---

## 八、积分规则

- **任务对应积分**：按难度 1/2/3/5 分档，验收通过才加分
- **一个任务只加一次分**：即使打回重做再通过，也不重复加（判重触发器）
- **撤销验收**：扣回已加的分
- **兑换**：孩子申请→爸妈同意→扣分兑付

---

## 九、打卡与历史

- **任务与打卡 1对多**：一个任务每天可多次打卡，每次存一条流水（时间/备注/照片/录音）
- **任务卡片「历史」按钮**：点开看该任务当天所有打卡记录，按时间倒序
- **统计页「打卡明细」**：看任意日期的所有打卡记录
- **打回重提交**：带出上次的备注/照片/录音，可删可改可加新

## 十、照片功能

- **拍照上传**：打卡时拍作业照，前端压缩（长边1280/JPEG 0.7）
- **查看**：点📷 → 列表 → 点单张全屏
  - 双指缩放（1-6倍）、单指平移、双击放大/还原
  - ⟳ 旋转90度（方向不对时转正）
  - ⬇ 下载到本地
  - ‹ › 切换上下张
- **录音**：打卡时录背诵，验收时点🎙播放

---

## 十一、多端同步与离线

- **多端同步**：爸妈孩子各自设备登录同一家庭，数据实时同步（Supabase Realtime）
- **离线**：断网时打卡暂存本地，联网自动同步（离线能力有限，建议联网使用）

---

## 十二、孩子模式说明

- 登录：选「我是孩子」→ 家庭名 + 选自己
- 能做：看今日清单、打卡（备注+拍照+录音）、看自己统计、申请兑换
- 不能做：验收、改设置、改积分、直接兑换
- 左上角显示自己名字（如"小修灏·准三年级"）
- 退出点右上「退出」

## 附录：技术架构

- **前端**：PWA（HTML+JS+CSS，Safari 添加主屏幕，无需应用商店）
- **后端**：Supabase（云数据库 + 实时同步 + Storage 存照片录音）
- **部署**：GitHub Pages（HTTPS）
- **数据**：云端存储，换设备登录同账号数据不丢
